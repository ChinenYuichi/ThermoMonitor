# ThermoMonitor 本番サーバー直接テスト手順書

対象バージョン：v2.9.3  
作成日：2026-05-05

---

## 概要

`csv02.exp.triam.kyushu-u.ac.jp` 上の `time.txt` を直接書き換えることで、hostsファイルの操作なしに完全な本番経路でのテストを行う。

| 項目 | 接続先 | 内容 |
|---|---|---|
| 温度データ | `192.168.52.248:502`（実機） | 実際のセンサー温度を取得 |
| ショット時刻 | `csv02.exp.triam.kyushu-u.ac.jp`（本番） | `time.txt` をSSHで直接書き換え |

hostsファイルの操作が不要で、ThermoMonitorが実際に使う経路をそのままテストできる。

---

## 事前準備（要・管理者確認）

以下の情報を管理者から入手すること。

| 必要な情報 | 内容 | 入手状況 |
|---|---|---|
| SSHユーザー名 | `csv02` へのログインアカウント | **確認済み（非公開）** |
| SSHパスワード / 鍵ファイル | 認証情報 | **確認済み（非公開）** |
| `time.txt` のサーバー上のパス | `/var/www/html/expinfo/time.txt` | **確認済み** |

---

## SSH接続確認

情報入手後、PowerShellで接続確認を行う。

```powershell
ssh <ユーザー名>@csv02.exp.triam.kyushu-u.ac.jp
```

接続できたら、`time.txt` の場所とパーミッションを確認する。

```bash
cat /var/www/html/expinfo/time.txt
ls -l /var/www/html/expinfo/time.txt
```

---

## テスト実行手順

### 1. ThermoMonitor を起動・接続

`ThermoMonitor.exe` を起動し、IPアドレス `192.168.52.248`・ポート `502` で接続する。

ログに以下が表示されることを確認する。

```
接続成功: 192.168.52.248:502
```

温度グラフ（4CH）が実際のセンサー値で更新されていることを目視確認する。

### 2. テストシーケンスの実行

PowerShellから `ssh` コマンドで `time.txt` を書き換える（別ウィンドウで実行）。

#### ステップ1：待機状態の確認

```powershell
ssh <ユーザー名>@csv02.exp.triam.kyushu-u.ac.jp "echo '-36000.0' > /var/www/html/expinfo/time.txt"
```

期待動作：ログに何も出力されない（待機中）

#### ステップ2：SHOT START の確認

```powershell
ssh <ユーザー名>@csv02.exp.triam.kyushu-u.ac.jp "echo '1.0' > /var/www/html/expinfo/time.txt"
```

期待動作：ログに `[SHOT START]` と実機からの開始温度が記録される

#### ステップ3：SHOT STOP の確認（数十秒後）

```powershell
ssh <ユーザー名>@csv02.exp.triam.kyushu-u.ac.jp "echo '-36000.0' > /var/www/html/expinfo/time.txt"
```

期待動作：ログに `[SHOT STOP]` と実機からのピーク温度が記録される

### 3. ログの確認

```powershell
Get-Content -Path C:\Users\quest\ThermoMonitor_log.txt -Tail 20
```

正常時の出力例：

```
[SHOT START] Shot time: 1.0 s, Start temp CH0: 35.2 ℃
[SHOT STOP]  Shot time: -36000.0 s, Peak temp CH0: 36.8 ℃
```

### 4. -12 エラーが出ないことの確認

ログに以下が**出ないこと**を確認する。

```
NON-FATAL RUN-TIME ERROR: "DisconnectFromTCPServer" ... Error Code: -12
```

## テスト後の後片付け

`time.txt` を元の値（待機状態）に戻す。

```powershell
ssh <ユーザー名>@csv02.exp.triam.kyushu-u.ac.jp "echo '-36000.0' > /var/www/html/expinfo/time.txt"
```

hostsファイルの操作は不要。

## 各テスト方法の比較

| 方法 | 温度データ | ショット時刻 | hostsファイル操作 | 本番経路 |
|---|---|---|---|---|
| ローカルテスト | ダミー | ローカル | 必要 | × |
| ハイブリッドテスト | **実機** | ローカル | 必要 | 一部 |
| **本番サーバー直接テスト** | **実機** | **本番** | **不要** | **完全** |
