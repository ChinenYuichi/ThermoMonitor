# ThermoMonitor 改修履歴

改修日: 2026-04-30
バージョン: v2.5.0 → v2.6.0

---

## 1. Q型センサタイプ追加（Y軸レンジ 0 ~ 150℃）

**依頼内容:** センサタイプ選択リングに「Q型」を追加し、Y軸レンジを 0℃（最低）〜 150℃（最高）に設定する。

---

### 変更ファイル一覧

| ファイル | 変更箇所 | 変更内容 |
|---|---|---|
| `modbus_tcp.h` | `SensorType` 列挙体 | `SENSOR_TYPE_Q = 0x07` を末尾に追加 |
| `ThermoMonitor_app.h` | `SENSOR_TYPE_COUNT` マクロ | `7` → `8` に変更 |
| `ThermoMonitor.c` | `SensorTypeNames[]` | `"Q型 (0 ~ 150℃)"` を末尾に追加 |
| `ThermoMonitor.c` | `SensorTypeSel_Callback` | Q型のY軸レンジ条件分岐を追加 |
| `ThermoMonitor.uir` | `PANEL_SENSOR_TYPE_SEL`（Ringコントロール） | Q型の選択肢を末尾に追加 |

---

### 変更詳細

#### ① `modbus_tcp.h` — `SensorType` 列挙体

```c
/* 変更前 */
    SENSOR_TYPE_T = 0x06
} SensorType;

/* 変更後 */
    SENSOR_TYPE_T = 0x06,
    SENSOR_TYPE_Q = 0x07
} SensorType;
```

#### ② `ThermoMonitor_app.h` — `SENSOR_TYPE_COUNT`

```c
/* 変更前 */
#define SENSOR_TYPE_COUNT   7

/* 変更後 */
#define SENSOR_TYPE_COUNT   8
```

#### ③ `ThermoMonitor.c` — `SensorTypeNames[]`

```c
/* 変更前 */
    "T型 (-100 ~ 400℃)"
};

/* 変更後 */
    "T型 (-100 ~ 400℃)",
    "Q型 (0 ~ 150℃)"
};
```

#### ④ `ThermoMonitor.c` — `SensorTypeSel_Callback` Y軸レンジ条件分岐

```c
/* 変更前 */
    else if (sType == SENSOR_TYPE_N) { yMin = -100.0; yMax = 1300.0; }

/* 変更後 */
    else if (sType == SENSOR_TYPE_N) { yMin = -100.0; yMax = 1300.0; }
    else if (sType == SENSOR_TYPE_Q) { yMin =    0.0; yMax =  150.0; }
```

#### ⑤ `ThermoMonitor.uir` — UIRエディタ作業

- `PANEL_SENSOR_TYPE_SEL` Ringコントロールを開き、末尾に `"Q型 (0 ~ 150℃)"` の選択肢を追加する。

---

### 注意事項

- デフォルト選択はK型のまま変更なし（`ThermoMonitor.c` の `SENSOR_TYPE_K` 設定はそのまま）
- `modbus_tcp~cvi102559.h` はCVIのバックアップファイルのため変更不要
- UIRファイル変更後、CVIが `ThermoMonitor.h` を自動再生成するため、ビルド前に内容を確認すること
